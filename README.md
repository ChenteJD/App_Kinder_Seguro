# Kinddo — Sistema Integral para Kinder Leizza

## 📌 Contexto Local y Perfil Socioeconómico

- Xicotepec de Juárez es una ciudad semi-urbana / semi-rural, con acceso moderado a internet móvil.
- Desigualdades sociales y económicas presentes: algunas familias con recursos limitados, otras con mayor estabilidad.
- Prácticamente todos los padres tienen al menos un celular con WhatsApp y redes sociales, por lo que una PWA sencilla y clara funcionaría bien.
- Fiabilidad del proyecto alta si se considera capacitación inicial y posibles zonas con señal deficiente.

---

## 🎯 Necesidades del Kinder Leizza

- Perfil de alumnos: datos médicos, emergencias, alergias.
- Comunicación directa con padres.
- Avisos y recordatorios de materiales, uniformes, excursiones y eventos.
- Gestión de pagos de colegiatura o cuotas.
- Reportes educativos y sociales (lenguaje, motricidad, convivencia).
- Registro de asistencia diaria y evidencias visuales (fotos).
- Perfil de maestros/pedagogos con certificaciones y roles definidos.
- Confirmación de “enterado” para avisos importantes.

## ⚡ Fiabilidad del proyecto

**Fortalezas:**
- Alta penetración de celulares.
- Familiaridad con apps simples reduce curva de aprendizaje.
- Relevancia para resolver problemas reales de comunicación.

**Desafíos / Barreras:**
- Zonas con mala señal de internet móvil.
- Celulares básicos podrían limitar uso de imágenes pesadas.
- Necesidad de capacitación inicial para maestros y padres.

> Con diseño simple, modo offline parcial y notificaciones ligeras, el proyecto es muy viable.
